# Proyecto 2 - Katas JavaScript

## Descripción

Este proyecto consiste en la realización y entrega de una serie de ejercicios (katas) de JavaScript correspondientes a distintos niveles de dificultad y temáticas del lenguaje.

El objetivo principal es reforzar y consolidar los conocimientos adquiridos durante el máster mediante la práctica constante y la resolución de problemas.

## Contenido del proyecto

El repositorio incluye ejercicios organizados en los siguientes bloques:

### Ejercicios básicos y avanzados de JavaScript
- Variables y tipos de datos
- Condicionales
- Bucles
- Funciones
- Arrays y objetos
- Destructuring y operadores

###  Ejercicios del DOM
- Manipulación del DOM
- Eventos
- Selección de elementos
- Creación y modificación de nodos
- Interacción con el usuario

### ES6+ y asincronía
- Arrow functions
- Promesas
- Async / Await
- Fetch API
- Manejo de datos asíncronos

## Tecnologías utilizadas

- JavaScript (ES6+)
- HTML5 (en ejercicios DOM)
- CSS3 (en ejercicios DOM)
- Node.js (opcional para ejecución)
- Git y GitHub
